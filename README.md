# Basic Python Calculator

This project is a simple calculator developed using Python.
It performs basic arithmetic operations such as addition, subtraction,
multiplication, and division.

## Features
- Menu-based user interface
- Modularized code with three main functions
- Handles invalid choices
- Error handling for division by zero

## Technologies Used
- Python 3

## How to Install & Run
1. Install Python on your computer
2. Download this repository
3. Open Command Prompt / Terminal in this folder
4. Run the command:

   python calculator.py
